# customer-segmentation-mlops

##  Objective
To segment customers based on their behavior using KMeans clustering.

##  Dataset
Sample customer dataset with:
- Age
- Annual Income
- Spending Score

##  Model
- Algorithm: KMeans Clustering
- Number of clusters: 3

##  Project Structure
data/        → dataset  
notebooks/   → analysis & training  
src/         → scripts (future use)  
models/      → saved model (.pkl)  
reports/     → outputs  

## Tools Used
- Python
- Pandas
- Scikit-learn
- Joblib
